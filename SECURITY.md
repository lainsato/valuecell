# Security Policy

## Supported Versions

We patch only what we still care about—and what *you* should still be running. 

| Version | Supported | Details |
|---------|-----------|---------|
| 0.1.x   | ✅        | Actively maintained. Critical/security fixes within **14 days** of validated report. |
